this project have a 3 taskes:
1- string compression task in file "Compression" 
2- thread safe task in file "ThreadSafe"
3- log standardizer in file "LogStandardizer"

------------------------------------------------------------

how to use :
. in Program.cs class there's a 3 section of code separated by comments //////.
. if you run the project all sections will run.
. the "LogStandardizer" will create an output file if it dose'nt exist.
. you can run the program by one click.
. the program start point (main) is Program.cs file.
